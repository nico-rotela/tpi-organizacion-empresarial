# TPI - Organización Empresarial

## Automatización del Proceso de Solicitud de Vacaciones

### Descripción

Este proyecto fue desarrollado para la materia Organización Empresarial de la Tecnicatura Universitaria en Programación a Distancia (UTN).

El objetivo consiste en automatizar el proceso de solicitud de vacaciones mediante un chatbot desarrollado en Python, basado en diagramas BPMN 2.0 que representan el proceso actual (AS-IS) y el proceso optimizado (TO-BE).

### Tecnologías Utilizadas

- Python
- Archivos CSV
- BPMN 2.0 (Bizagi Modeler)

### Estructura del Proyecto

- chatbot.py: programa principal.
- empleados.csv: información de empleados y días disponibles.
- solicitudes.csv: registro de solicitudes realizadas.
- Informe_TPI.pdf: documentación del proyecto.
- Diagramas BPMN AS-IS y TO-BE.

### Ejecución

1. Descargar o clonar el repositorio.
2. Verificar que Python esté instalado.
3. Ejecutar el archivo principal:

```bash
python chatbot.py
```

4. Seguir las instrucciones mostradas por el chatbot.

### Funcionalidades

- Validación de empleados mediante legajo.
- Consulta de días disponibles.
- Registro de solicitudes.
- Aprobación o rechazo automático.
- Manejo de errores de entrada.
